ETL Project with Python

A Python-based ETL pipeline project using SQLite3, Requests, Pandas, and BeautifulSoup to extract and process country-wise GDP data from the IMF. The data is stored in a JSON file and a database table. The script includes a query to filter countries with GDPs over 100 billion USD and logs the entire process. This script ensures automated biannual updates for efficient data access.
